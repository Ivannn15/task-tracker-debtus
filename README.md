# Task Tracker API

Task Tracker API — это демонстрационный сервис управления задачами сотрудников. Решение реализовано на ASP.NET Core 8 с использованием Entity Framework Core и SQLite и соответствует требованиям тестового задания.

## Основные возможности

- Управление сотрудниками, проектами, группами задач и задачами.
- Фильтрация задач по исполнителю/наблюдателю, группе и проекту.
- Контроль состояний задач с жёстко заданной последовательностью переходов и защитой от гонок данных с помощью токена конкурентного доступа.
- Настройка состава исполнителей и наблюдателей с проверкой на взаимоисключающие роли.
- Автоматическая инициализация базы данных тестовыми данными (минимум 20 записей для сотрудников, проектов, групп и задач).
- Единый формат ошибок в стиле `application/problem+json`.
- Документация и тестирование через встроенный Swagger UI.

## Запуск проекта

```bash
cd TaskTracker.Api
dotnet run
```

При первом запуске база данных `tasktracker.db` будет создана автоматически и заполнена тестовыми данными.

Swagger UI доступен по адресу `http://localhost:5000/swagger` (порт уточняется в выводе `dotnet run`).

## Структура решения

- **TaskTracker.Domain** — доменные модели и перечисления.
- **TaskTracker.Application** — DTO и контракты сервисов прикладного уровня.
- **TaskTracker.Infrastructure** — реализация сервисов, контекст EF Core, конфигурации и сидирование данных.
- **TaskTracker.Api** — слой представления (Web API), обработка ошибок и конфигурация DI.

## Тестовые данные

Сидирование данных включает 20 сотрудников, 20 проектов, 20 групп задач и 20 задач со связанными исполнителями и наблюдателями. Статусы задач распределены по всем стадиям конвейера, что позволяет проверить все сценарии API.

## Дополнительные замечания

- Для хранения данных используется SQLite. При необходимости подключение можно изменить в `appsettings.json` (строка подключения `ConnectionStrings:TaskTracker`).
- Все зависимости регистрируются через `AddInfrastructure`, что упрощает замену реализации сервисов и повторное использование слоёв приложения.
